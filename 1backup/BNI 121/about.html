How to save token cost

Five tools every Bettroi developer installs on day one. Used together they cut Claude Code spend by 50 to 90 percent without changing how you prompt.

Tool Purpose Key benefit
1. Knowledge Graph (Graphify)
 Maps the entire codebase visually and logically. Saves massive tokens by helping Claude "ride the lines" to the right file instead of searching blindly.
2. Firecrawl MCP
 A "strainer" for the web that turns messy HTML into clean data. Scrape leads and docs without wasting tokens on ads, banners, and cookie pop-ups.
3. NotebookLM Skill
 Connects Claude to Google's research tool via browser cookies. Query thousands of sources (PDFs, YouTube videos) at $0 retrieval cost.
4. Awesome Design
 A library of 68 brand identities (Apple, Claude, Linear, etc.). Gives Claude "taste" so it generates gorgeous, modern UI instead of generic-looking code.
5. Claude Code Router
 A local proxy that routes tasks to different models via OpenRouter. Routes simple tasks to cheap models (DeepSeek), heavy tasks to Opus, saving up to 80% on API costs.

Three-brain switch

Three terminal commands let you flip Claude Code between providers per session:

  • claude-ds  DeepSeek (cheap)
  • claude-glm Z.AI / GLM (alt cheap)
  • claude-an  Anthropic (premium)
  • claude-status which brain is active

Cost difference between cheap brains and Anthropic is roughly tenfold for similar work.

Daily flow

  • Run /graphify on any unfamiliar repo before editing.
  • Use firecrawl_search instead of plain WebFetch for any URL.
  • Stash long PDFs and YouTube transcripts into NotebookLM, query at $0.
  • Tell Claude "set active design to <brand>" before generating UI.
  • Routine boilerplate? Launch via ccr code for DeepSeek.

Install in one place

github.com/chatgptnotes/claude-code-deepseek-backend
